## Getting Started

Follow the instructions below to clone and explore the repository on your local setup.

### Prerequisites

Ensure you have the following tools and credentials ready:
- A Databricks account
- Databricks CLI installed and configured
- Python environment (Python 3.8+ recommended)
- Git installed on your system

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aviral-bhardwaj/databricksLearning.git
   cd databricksLearning
   ```

2. Set up the Python environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # For Linux/macOS
   # OR
   venv\Scripts\activate     # For Windows
   pip install -r requirements.txt
   ```

3. Configure Databricks CLI:
    ```bash
    databricks configure --token
    ```

4. Run notebooks or examples by uploading them to your Databricks workspace or running locally.
